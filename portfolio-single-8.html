Travel Website

Scroll

Project Date November 27th, 2025
Role Digital Designer
Platform Wordpress
Visit https:///
Overview

Holiday Asia is a licensed travel agency registered with the Ministry of Tourism, Malaysia. The website is designed to help users easily explore destinations, tour packages, and travel services, including flights, hotels, group tours, and lifestyle experiences, with a focus on clarity, accessibility, and smooth trip planning.

Problem

• Navigate a large number of tour packages and destinations
• Access supporting materials (itineraries, PDFs, language versions) without confusion
• Navigate a large number of tour packages and destinations

Solution

• Organize content using clear categories, filters, and destination hierarchy
• Surface only relevant content based on user selection
• Conditionally display elements (e.g. itinerary buttons) to reduce visual noise

Design Approach

• User-centred design focusing on clarity and ease of use
• Consistent UI components for clarity
• Responsive design for desktop and mobile experience
